# Dados para Análise

Este projeto utiliza duas bases de dados distintas:
- **Bike** Conjunto de dados bruto, sem formatação.

-  **Work** Conjunto de dados tratado e formatado para análise.
 
 ## Observações 
 - A base **Bike** serve como ponto de partida, contendo os dados originais.
 - A base **Work** é derivada da anterior, já estruturada para facilitar o processo de análise.
